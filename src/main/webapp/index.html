How to mosaiking

First, selectt the image you want to mosaicing. Selection method are two types. The first type is how to choose from a file. The secont type is how to select the photos you took with the camera.

Then, set the option of mosaicing. Setting items are division number of horizontal, division number of vertical and keyword of the image to be used in mosaicing.

Finally, please press the send button. As soon as mosaicing is finished, the results are displayed at the bottom.

So, enjoy the MosaicA!!
